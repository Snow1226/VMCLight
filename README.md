## VMCLight
VMCProtocolを使用して、BeatSaberのライトをVirtualMotionCaptureへ送信します。  

### 対応ゲームバージョン
BeatSaver 1.40.8

### 必要Mod
BSIPA 4.3.6  
BSML 1.12.5  
Chroma 2.9.20  

### Install
[Release](https://github.com/Snow1226/VMCLight/releases)からダウンロードし、
zipファイルの中身をBeatSaberフォルダへ入れてください。

### 設定
VMCProtocol Address　: VMCProtocol送信先のアドレス。同じPC内は127.0.0.1 を使用してください。  
VMCProtocol Port : VMCProtocol送信先のポート。VirtualMotionCaptureの受信ポートでLightが有効になっているのを確認してください。  
Blend Color : BeatSaberのライト色をそのまま使用するとアバターがやや暗めになるので、デフォルトでは白を混ぜます。  
Blend Color Intensity : 上記Blend Colorを混ぜる強度を指定します。0で混ぜません。1でBlendColorになります。
